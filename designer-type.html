10 Designer Type

Find your designer type by locating the letter combination from the assessment results!

Identification

EFRO

  • Cognition: People driven, open minded, decisive
  • Behavior: Always seeking for innovative and different solutions, productive working in teams.
  • Example: System Designer, Civil Engineers

ECRO

  • Cognition: Task-oriented, innovation-driven
  • Behavior: Working in group for innovation according to guidelines and criterias
  • Example: Mechanical Engineer (designer), Consultant

EFRI

  • Cognition: Understanding problems based on existing facts, embracing creative solutions
  • Behavior: Flexible with guidelines, driving innovative proces
  • Example: Architect, Branding Strategist

ECRI

  • Cognition: Interested in (technology) innovation
  • Behavior: Searching for concrete evidence to develop creative solutions
  • Example: Game designer, IT Entrepreneur

Ideation

SFRI

  • Cognition: Ambitious, visionary individuals, generating unicorn ideas
  • Behavior: Working alone, use words to persuade, passionate speakers who takes on challenge
  • Example: Entrepreneur, Inventor

SCRO

  • Cognition: Finding breakthrough, creative thinking
  • Behavior: Collaborator, solving defined tasks and problems from unique perspectives
  • Example: R&D, Strategis

SFRO

  • Cognition: Creative, value innovation
  • Behavior: Insights-driven, brainstorming solutions, working in groups
  • Example: Design Consultant, Project Manager

SCRI

  • Cognition: Concept-driven
  • Behavior: Commissioned or contracted, creating alone
  • Example: Studio Owner, Product Designer, Freelancer

Selection

SFVO

  • Cognition: Using intuition for decision making, value detail improvements
  • Behavior: Embracing uncertainty and changes, working in iterative proces
  • Example: Industrial Designer, Prototyper

SFVI

  • Cognition: Empathetic individuals who can easily relate to other peopleâ€™s problem
  • Behavior: Selecting the optimized solution and building upon the current solution
  • Example: UX Researcher, Anthropologist, Sculpt

SCVI

  • Cognition: Creative but value iterative process, efficient problem solver at given tasks
  • Behavior: Embracing creativity, more comfortable working with guidelines, independent, iterative
  • Example: UI Designer, Graphic Designer, Front-End Develop

SCVO

  • Cognition: Critical thinking, value improvements and guidelines
  • Behavior: Team player, working on developed products and ideas, but still embracing creativity
  • Example: Marketing Manager, Architect (concept)

Implementation

EFVI

  • Cognition: With goal in mind, achievement-driven
  • Behavior: Working alone, searching for evidence
  • Example: Market Analyst, Information Architect

ECVI

  • Cognition: Detailed and analytical individuals, mission-oriented
  • Behavior: Refining details, iterative, interested in validation and testing
  • Example: Product Developer, Data Scientist, Program Developer

ECVO

  • Cognition: Analytical thinking, details-oriented
  • Behavior: Working with data and requirements, working collaboratively
  • Example: System Engineer, Mechanical Engineer

EFVO

  • Cognition: Facts-driven, comfortable for modifying requirements
  • Behavior: Working in iterative improvements, always searching for validation
  • Example: UX Designer, System Designer
